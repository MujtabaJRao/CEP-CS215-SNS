# 🎸 Digital Guitar Sound Synthesis (Karplus–Strong)

## 📌 Overview
This project focuses on the modeling and implementation of a digital guitar sound synthesis system using the Karplus–Strong algorithm. It applies core concepts from Signals and Systems to simulate realistic plucked string sounds using discrete-time signal processing techniques.

---

## 🎯 Objectives
- Model a plucked string mathematically  
- Implement the Karplus–Strong algorithm  
- Generate realistic guitar-like audio  
- Apply theoretical DSP concepts in a practical system  

---

## ⚙️ How It Works
- A burst of random noise simulates the string being plucked  
- The signal is passed through a delay line (buffer)  
- A feedback loop sustains the sound  
- A simple averaging filter causes gradual decay  

**Flow:** Noise → Delay → Feedback → Decay → 🎸 Sound
